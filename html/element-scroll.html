Scroll of the Element

  • element.scrollTop - получает или устанавливает количество пикселей, на которое контент элемента прокручен вверх.
  • element.scrollLeft - получает или устанавливает количество пикселей, на которое контент элемента прокручен влево.

function next() {
  element.scrollTop = img.height;
