Методы смещения документа

  • window.scrollBy(X, Y, option) - смещение в px документа относительно текущего положения. Допустимы отрицательные значения.
  • window.scroll(X, Y, option) - перемещение в px к заданым координатам документа document (абсолютное смещение)
  • option для этих методов:
    • top : координата Y
    • left : координата X
    • behavior :
      • auto по умолчанию
      • smooth
      • instant
